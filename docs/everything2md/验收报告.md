# Everything2MD 项目验收报告

## 项目概述

Everything2MD 是一个将各种办公文档格式转换为 Markdown 格式的工具。该项目支持多种文件格式，包括 Microsoft Word、PowerPoint、PDF、文本文件和 Markdown 文件。

## 验收检查

### 所有需求已实现

- [x] 支持 Microsoft Word 文档（.doc, .docx）转换
- [x] 支持 Microsoft PowerPoint 演示文稿（.ppt, .pptx）转换
- [x] 支持 PDF 文件转换
- [x] 支持文本文件（.txt）转换
- [x] 支持 Markdown 文件（.md, .markdown）处理
- [x] 命令行界面支持
- [x] 详细日志输出功能
