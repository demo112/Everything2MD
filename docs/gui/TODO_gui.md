## 有用的指引

### 1. 启动GUI程序
```bash
# 在项目根目录下执行
python3 src/gui/main.py
```

### 2. 开发环境搭建
```bash
# 确保Python 3.6+已安装
python3 --version

# 通常Tkinter会随Python安装，如未安装可尝试:
# Ubuntu/Debian:
sudo apt-get install python3-tk

# macOS (使用Homebrew):
brew install python-tk
```

### 3. 代码结构说明
- `src/gui/main.py`: GUI主程序文件
- `docs/gui/`: GUI开发相关文档目录
  - `Align阶段/`: 需求分析文档
  - `Architect（架构阶段）/`: 架构设计文档
  - `Atomize（原子化阶段）/`: 任务分解文档
  - `Approve（审批阶段）/`: 审批检查清单
  - `Automate（自动化执行）/`: 开发过程文档
  - `Assess（评估阶段）/`: 验收测试文档

### 4. 贡献指南
1. Fork项目仓库

### 5. 故障排除
- 如果GUI无法启动，检查Python和Tkinter是否正确安装
- 如果转换功能异常，检查Everything2MD核心脚本是否正常工作
- 如果界面显示异常，检查系统显示设置和字体配置
